NPM (Node Package Manager)

What is it?

NPM is a registry for open-source code packages

You can look up all packages here

The NPM CLI or Command Line Client is installed along with your installation of Node

You can test if NPM is installed using the command:

npm --version

To start installing NPM package on your Node.js project simply create a file called:

package.json

NPM uses this file to store information about your project package, such as dependencies.
This is a JSON file which means that it contains attribute-value pairs.

A the interior of a typical package.json would look something like this:

{
     "dependencies": {
         "express": "^4.17.1"
     }
}

Here the "dependencies" contains attribute-value pairs that describe which NPM packages the project is using and which version.
in this case: the project uses the NPM package "express"

You can also skip creating your own package.json file and let NPM do it for you, with the command:

npm install express

The usage of a package.json file means that you wont have to add your dependencies when you push your project with git
which saves your project from becoming bloated.
