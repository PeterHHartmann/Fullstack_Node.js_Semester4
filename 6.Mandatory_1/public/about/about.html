What is Node.js

Node.js is JavaScript runtime environment making it possible to write and run JavaScript code on the server-side, as opposed to the browser/client-side which JavaScript was originally intended for. Node run on the V8 engine, an open-source JavaScript engine which compiles JavaScript to native machine code.
