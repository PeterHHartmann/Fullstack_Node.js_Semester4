Nodemon

What is it?

Nodemon is a CLI tool which helps speed up the development process
by restarting your application every times it detects changes

It is installed with NPM with the command:

npm install -g nodemon

Now instead of using the usual "node <your-application>"
you can instead start your application with:

nodemon <your-application>

Nodemon should now have started your application and will restart if you make any JavaScript changes in your project
