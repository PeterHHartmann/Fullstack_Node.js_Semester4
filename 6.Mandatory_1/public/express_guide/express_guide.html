Express

Express is web framework for Node, which allows you to quickly and easily setup a webserver in your project

The first step of express is to create a new JavaScript file such as "app.js"
You then want to create a constant with which you import express:

const express = require('express')
const app = express()

The require function is built into Node as a way of including modules that exist in seperate files, like an express installation (which can be found in the node_modules folder in your project)
