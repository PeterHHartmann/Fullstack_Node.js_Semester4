Variables and Datatypes

var, let and const
var variable keyword can be redeclared and updated.

let variable keyword be be updated but not redeclared.

const variable keyword cannot be updated.

JavaScript is a loosely typed and dynamic language.

This means that variables are not directly associated with a type, and can be assigned and re-assigned values of all types.

Strings

String can be declared with but single and double quotations:

const myString = 'Hello World!';

const myString = "Hello World!";

to convert a String to a number you can use the Number() constructor:

const numberOne = "1.10";

const numberTwo = "2.30";

const combined = Number(numberOne) + Number(numberTwo);

Objects

A JavaScript objects is a collection of properties. A property is an association between a name (or key) and a value:

const me = {
    name: "Peter Helle Hartmann",
    age: 25
};

keys are seperated from values with a colon (:)
and each property is comma seperated

Arrays

Arrays in Javascript differ from arrays in languages such as Java in by the fact that an array isn't of a specific type,
and can therefore contain different types such as numbers and Strings:

const array = ["Hello", 42, "This is an array"];

Inserting new values into arrays are done using the push() function

Traversing an array can be done with a simple for loop
but is most optimally done using these methods on the array depending on why you need to traverse:

map()
filter()
find()

Functions

JavaScript functions looks like this:

function getGreeting (){
     return "greetings my friend";
};

Functions can also accept other functions as an argument, these are called callback functions:

function sayHiLater(anyFunctionReference) {
     anyFunctionReference();
};

JavaScript also supports anonymous functions which uses the arrow syntax:

const pokePerson = (name) => {
     return "Poke " + name;
};
