Serving HTML files

Basics

Serving HTML files is easy with express

First simply define a get method with the desired url

app.get("/serving_html", (req, res) => {
     res.sendFile(__dirname + "/public/serving_html/serving_html.html");
});

notice that we use the sendFile() function here instead of the previous send()
__dirname is an environment variable that tells you the absolute path of the directory containing the currently executing file.

Static

if the HTML files you are sending to the client are using other files such as images, css, or client-side JavaScript,
you can include a directory in which these files are stored to be served statically.
Just add this line of code to your code

app.use(express.static('public'))

where 'public' is the name of the folder containing the files to be served statically
